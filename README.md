# 🚦 Smart Traffic Monitoring & Prediction System

## 📌 Overview
This project uses YOLOv5 and OpenCV to detect vehicles and dynamically adjust traffic signal timing.

## 🧠 Features
- Vehicle detection using YOLOv5
- Real-time traffic density estimation
- Dynamic green signal timing

## 🛠 Tech Stack
- Python
- OpenCV
- PyTorch
- YOLOv5

## ▶️ How to Run

pip install -r requirements.txt  
python main.py --source traffic.mp4

## 📊 Output
- Displays vehicle count
- Calculates green light time dynamically
